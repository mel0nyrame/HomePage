# 项目文档

## 项目描述

该项目为展示自制游戏想法的交流平台和官网

## 实现功能

- 需要实现的前端功能
  - 渲染md文档格式

- 需要实现的后端功能
  - 用户登陆，登出
  - 论坛交流，发布帖子，给帖子打标签
  - 帖子浏览量、点赞量、曝光量
  - 搜索帖子，支持模糊搜索，过滤搜索
  - 用户私信
  - 用户更换用户名，邮箱，查看自己发过的帖子
  - 拉黑用户

## 业务逻辑

### 用户登陆、鉴权

使用Spring Security + JWT + Redis处理登陆业务

1.在SecurityConfig配置中限制处理login之外的所有页面都要强制登陆
2.在JwtAuthenticationFilter拦截器中校验登陆的token，并将token放进Redis中
3.在Controller和Service中使用SecurityContextHolder获取用户信息

示意:

```text
[客户端] 
    │
    │ (1) POST /login (用户名密码) -> AuthController.login() -> 返回JWT
    │
    │ (2) GET /user/me (Header: Bearer <JWT>)
    ▼
[Filter层] JwtAuthenticationFilter 
    ├── 解析Token、校验签名、有效期
    ├── 从Redis/Token获取用户信息
    └── SecurityContextHolder.setAuthentication(authentication)  // 认证完成
    │
[Controller层] UserController.getMe()
    └── @AuthenticationPrincipal UserDetails user  // 直接拿到用户，处理业务
[Service层] UserService.getCurrentUser()
    └── SecurityContextHolder... 同样能拿到，不需要Token
```
